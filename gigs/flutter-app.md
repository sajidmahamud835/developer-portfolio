### About This Gig

Get a **native-quality iOS and Android app** from a single codebase using Flutter. Smooth animations, offline support, and API integration.

Flutter allows for rapid development without compromising on performance or look and feel.

#### Features:
*   Cross-Platform (iOS & Android)
*   Custom UI Widgets
*   State Management (Provider/Bloc/Riverpod)
*   API Integration
*   Google Maps / Location Services
